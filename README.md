# Portfolio de Mario Gutierrez

Portfolio de desarrollador Full Stack e Ingeniero Mecatrónico, construido con Astro, Three.js y Vanilla CSS.

## Tecnologías

- **Framework**: [Astro](https://astro.build)
- **Gráficos 3D**: [Three.js](https://threejs.org)
- **Estilos**: Vanilla CSS (Variables CSS, Glassmorphism)
- **Fuentes**: Inter (Google Fonts)

## Instalación y Uso

1.  Instalar dependencias:
    ```bash
    npm install
    ```

2.  Iniciar servidor de desarrollo:
    ```bash
    npm run dev
    ```

3.  Construir para producción:
    ```bash
    npm run build
    ```

## Estructura

- `src/components/Background3D.astro`: Escena de Three.js con partículas y animaciones.
- `src/layouts/Layout.astro`: Layout principal con configuración de fuentes y estilos globales.
- `src/styles/global.css`: Definiciones de tema oscuro, variables y utilidades.
- `src/pages/index.astro`: Página principal con secciones de Biografía, Skills y Contacto.
